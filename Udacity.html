HTML

HTML stands for Hypertext Markup Languange and is used to create Webpages. A HTML document contains text content which defines what you will see including, text,video, images and etc and how the webpages looks like.

Tags and Elements

HTML documents used elements to tell the browsers the type of each elements by using TAGS

Below are the example of tags i learned so far.

< b > tag = Makes the 'text' content bold
< em > tag = Makes the 'text' content italic
< p > tag = defines a paragraph.
< span > tag = used to group inline-elements in a document.
< Strong > tag = is a phrase tag. It defines important text.
< img > tag = defines an image in an HTML page , has two required attributes: src and alt.
< a > tag = defines a hyperlink, which is used to link from one page to another.
< br > tag = inserts a single line break.

There are many others tags which can be found in this web site

Inline Vs Block elements

Every HTML element has a default display value depending on what type of element it is. The default display value for most elements is block or inline.
A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).
An inline element does not start on a new line and only takes up as much width as necessary.
